MARCOS GARCIA AUTHOR SITE — DRAG-AND-DROP UPDATE

WHAT THIS PACKET DOES
1. Adds the new “Universe’s Grand Mysteries” cover to the In Development section.
2. Adds the cosmic Marcus Aurelius bust beneath the author photo in Meet the Author.
3. Preserves your existing style.css, script.js, and existing images.

UPLOAD STEPS
1. Open:
   https://github.com/MarcosAurelious/marcos-garcia-author

2. Click “Add file” → “Upload files.”

3. Open this packet folder and drag these items into the GitHub upload area:
   - index.html
   - marcus-additions.css
   - images folder

4. GitHub should show:
   - index.html (replaces the existing version)
   - marcus-additions.css (new)
   - images/universe-grand-mysteries-cover.webp (new)
   - images/marcus-cosmic-bust.webp (new)

5. In the commit message, enter:
   Add new book cover and Marcus artwork

6. Click “Commit changes.”

7. Wait 1–3 minutes, then refresh:
   https://marcosaurelious.github.io/marcos-garcia-author/

IMPORTANT
- Do not delete your existing style.css, script.js, or other images.
- Upload the folder contents while keeping the “images” folder structure.
- GitHub filenames are case-sensitive.
- The current cover artwork reads “UNIVERSES GRAND MYSTERIES.”
  The website text uses “THE UNIVERSE’S GRAND MYSTERIES.”
  You may later replace the image with a corrected cover using the same filename.
